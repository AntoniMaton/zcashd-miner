## Installation

```
virtualenv .env
. .env/bin/activate
pip install git+https://github.com/ondrejsika/zcashd-miner.git
```

## Usage


```
zcashd-miner.py --help
```

Example of usage

```
zcashd-miner.py http://zu:zp@ram5.bo:28232 1
```
